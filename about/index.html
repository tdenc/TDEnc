何これ？

つんでれんこって？

  • ニコニコやYouTubeに動画をアップロードしたい
  • アップロードしたいけど、エンコが面倒、楽をしたい
  • 楽をしたいけど、それでもやっぱり綺麗に仕上げたい
  • そんなあなたに、つんでれんこ（詳しくはWebで！）

どんな人向けなの？

以下のような人を対象者に想定。

  • エンコード？ナニソレオイシイノ？
  • オプション多すぎメンドクセ。いちいち変えるのマンドクセ。
  • ぐぐーって持ってきて、ぽとっと落とせば、ぎゅいーんってエンコしてくんない？
  • ツンデレはぁはぁ＾ｑ＾

内部で何してるの？

以下の諸ツール・ライブラリ（バイナリ）を利用して、質問への返答を元にオプションを自動で調整してMP4を作成。CUIです。

  • Frame Server: Avisynth
  • Video Encode: x264
  • Audio Encode: qtaacenc(QuickTime), neroAacEnc
  • Mux: MP4Box

うちのパソコンで動く？

以下の環境での動作確認・報告があります（推奨環境：Windows 7以降）

  • Windows 2000
  • Windows XP SP3
  • Windows Vista 32bit & 64bit
  • Windows 7 32bit & 64bit
  • Windows 8/8.1 32bit & 64bit
  • Windows 10 64bit
